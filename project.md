# Achoo! - Aquarium Simulation Game

An aquarium simulation game where you feed fish, watch them grow, collect eggs for points, and build your underwater family.

## Status: Complete MVP ✅

## Quick Start

```bash
npm install
npm run dev
```

## Implemented Features

- [x] Click-to-feed mechanic (costs points)
- [x] Food pellets that sink through water
- [x] Fish AI with seeking behavior for food
- [x] Fish growth system (Baby → Juvenile → Adult → Elder)
- [x] Hunger/starvation mechanics
- [x] Egg production by adult fish
- [x] Click-to-collect eggs
- [x] Points economy system
- [x] Buy new fish button
- [x] Game over when all fish die
- [x] Stats tracking (max points, fish raised, eggs collected)
- [x] Beautiful underwater visuals (seaweed, bubbles, gravel)
